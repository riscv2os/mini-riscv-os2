可在 QEMU 測試的範例
1. UART 接收中斷 (推薦)
QEMU virt machine 有 UART，可測試字元輸入：
- 使用 -serial 或 -nographic
- 註冊 UART RX 中斷
- 當 QEMU 傳送字元時觸發 ISR
2. VirtIO 網路中斷
virt machine 支援 VirtIO 網卡：
- 封包到達時觸發中斷
- 可做簡單的網路接收處理
3. RTC 定時中斷
QEMU 支援 goldfish_rtc：
- 設定鬧鐘時間
- 到達時觸發中斷
